O que é HTML?

HTML significa HyperText Markup Language, é a linguagem padrão usada para criar e estruturar páginas na web.
Criada por Tim Berners-Lee, o HTML permite que você organize e formate o conteúdo de uma página, como textos, imagens, links, e outros elementos.


Sobre Title

Os elementos HTML <h1> até <h6> representam seis níveis de título de seção, onde <h1>é o nível de seção mais alto e <h6> é o mais baixo.
O seguinte código apresenta todos níveis de Cabeçalho em uso: 


         

         <h1>Esse é um exemplo de Cabeçalho nível 1</h1> 

         <h2>Esse é um exemplo de Cabeçalho nível 2</h2> 

         <h3>Esse é um exemplo de Cabeçalho nível 3</h3> 

         <h4>Esse é um exemplo de Cabeçalho nível 4</h4> 

         <h5>Esse é um exemplo de Cabeçalho nível 5</h5> 

         <h6>Esse é um exemplo de Cabeçalho nível 6</h6>

Sobre Listas

Listas não ordenadas:

Para criar uma lista não ordenada em HTML, você pode usar as tags <ul> e <li>.
A tag <ul> inicia a lista não ordenada e cada item da lista é definido dentro de uma tag <li>.
Podemos adicionar quantos itens quisermos dentro da lista, e eles serão exibidos com marcadores (bolinhas) por padrão.
Exemplo de código: 


         <ul>

         <li>Item 1</li>

         <li>Item 2</li>

         <li>Item 3</li>

         </ul>
Resultado deste código:

  • Item 2

Listas ordenadas:

Para criar uma lista ordenada em HTML, você pode usar as tags <ol> e <li>.
A tag <ol> inicia a lista ordenada e cada item da lista é definido dentro de uma tag <li>.
Os itens da lista serão numerados automaticamente pelo navegador.
Exemplo de código: 


         <ol>

         <li>Item 1</li>

         <li>Item 2</li>

         <li>Item 3</li>

         </ol>
Resultado deste código:

         
    
         
  1. Item 1
    2. 
         
  2. Item 2
    3. 
         
  3. Item 3
    4.
Podemos personalizar o tipo de numeração usando o atributo type na tag <ol>, como type="A" para letras maiúsculas ou type="a" para letras minúsculas.
Exemplo de código: 

         <ol type="A">

         <li>Item 1</li>

         <li>Item 2</li>

         <li>Item 3</li>

         </ol>
Resultado deste código:

         
    
         
  1. Item 1
    2. 
         
  2. Item 2
    3. 
         
  3. Item 3
    4.

Listas de menu:

O Elemento HTML de Navegação <nav> representa uma seção de uma página que aponta para outras páginas ou para outras áreas da página, ou seja, uma seção com links de navegação.
A tag é usada como um contêiner para a lista de itens de navegação.
Além da tag navegação <nav>, usaremos a tag de lista <ul> e a tag de âncoras <a>
Exemplo de código: 


         <nav>

         <ul>

         <li><a href="inicio.html">Home</a></li> 

         <li><a href="servicos.html">Serviços</a></li> 

         <li><a href="contato.html">Contato</a></li> 

         </ul>

         </nav>
Resultado deste código:

Sobre Âncoras

O elemento <a> em HTML (ou elemento âncora), com o atributo href="#" cria-se um hiperligação nas páginas web, arquivos, endereços de emails, ligações na mesma página ou endereços na URL.
O conteúdo dentro de cada <a> precisará indicar o destino do link.
Exemplo de código: 


         <h3>Índice<h3>

         <li><a href="#nome" title="clique aqui para ir ao destino NOME ESCOLHIDO">Nome escolhido</a></li>
Para âncorar usamos a tag id="" 

         <h2 id="nome">Nome escolhido</h2>

    Citações, Blockquote ou Q

    Tag Blockquote:

    O elemento <blockquote> é usado para indicar uma citação longa ou um trecho citado de outra fonte.
    Exemplo de código: 

    
         <blockquote>Este é um exemplo de citação longa, onde o ( blockquote ) cria um espaço no parágrafo para diferenciar.</blockquote>
    Resultado deste código:
     
         
    
         | Este é um exemplo de citação longa, onde o ( blockquote ) cria um espaço no parágrafo para diferenciar.

    Tag Q:

    O elemento <q> é utilizado para citações curtas. Diferente do <blockquote>, ele não cria uma nova linha ou parágrafo, mas adiciona aspas ao redor do texto citado.
    Exemplo de código: 

    
         <q>Este é um exemplo de citação curta, onde o ( q ) cria aspas para diferenciar.</q>
    Resultado deste código:
    
         Este é um exemplo de citação curta, onde o ( q ) cria aspas para diferenciar.

    Tag Cite:

    O elemento <cite> é utilizado para representar um referêcia a um trabalho artistico ou criativo. Isso pode incluir livros, artigos, filmes, músicas, entre outros.
    O conteúdo dentro da tag <cite> por padrão, é exibido em itálico.
    Exemplo de código: 

    
         Este é um exemplo de citação, onde o <cite>( cite ) escreve em itálico para diferenciar.</cite>
    Resultado deste código:
    
         Este é um exemplo de citação, onde o ( cite ) escreve em itálico para diferenciar.


    O que é a tag Hr?

    O elemento HTML <hr /> representa uma quebra temática entre elementos de nível de parágrafo, gerando uma linha.
    Exemplo de código: 

    
         <p><b>Este é um exemplo de parágrafo 01</b></p>
    
         <hr />
    
         <p><b>Este é um exemplo de parágrafo 02</b></p>
    Resultado do código:
    
        
    Este é um exemplo de parágrafo 01
    
        
    
        
    Este é um exemplo de parágrafo 02


    O que é a tag ABBR?

    O Elemento HTML <abbr> (ou Elemento de Abreviação HTML) representa uma abreviação e fornece uma descrição. Junto com a tag title, deve conter a descrição completa.
    Exemplo de código: 

    
         <p>O Elemento HTML<abbr title="Elemento de Abreviação HTML">abbr</abbr></p>
    Note que ao passar o mouse em cima da palavra abbr, aparece a frase de descrição.

    O que é Semântica?

    Durante muitos anos o elemento padrão no HTML era a div, construíamos nosso conteúdo todo baseado nela, e assim nascia a sopa de divs.
    Mas em 2014 saiu a quinta versão do HTML, e com ela vieram vários mudanças importantes, como performance e acessibilidade.
    E temos vários elementos para ressignificar as divs:

    <header> É o cabeçalho da página ou de uma seção da página e normalmente contém logotipos, menus, campos de busca.

    <section> Representa uma seção genérica de conteúdo quando não houver um elemento mais específico para isso.

    <article> Representa um conteúdo independente e de maior relevância dentro de uma página, como um post de blog, uma notícia em uma barra lateral ou um bloco de comentários.
    Um article pode conter outros elementos, como header, cabeçalhos, parágrafos e imagens.

    <aside> É uma seção que engloba conteúdos relacionados ao conteúdo principal, como artigos relacionados, biografia do autor e publicidade.
    Normalmente são representadas como barras laterais.

    <footer> Esse elemento representa o rodapé do conteúdo ou de parte dele, pois ele é aceito dentro de vários elementos, como article e section e até do body.
    Exemplos de conteúdo de um <footer> são informações de autor e links relacionados.

    <h1> - <h6> Eles não foram criados na versão 5 do HTML e nem são específicos para semântica, mas servem para esse propósito.
